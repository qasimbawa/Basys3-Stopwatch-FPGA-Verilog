# Basys3-Stopwatch-FPGA-Verilog

Programmable stopwatch/timer implemented in Verilog on Basys3 FPGA with FSM control, datapath architecture, clock division, and multiplexed seven-segment display.

Features
- Count up / count down modes
- Preset timer mode
- Start/stop/reset controls
- Four-digit seven-segment display
- FSM + datapath architecture

Tools
- Verilog
- Vivado
- Basys3 FPGA
